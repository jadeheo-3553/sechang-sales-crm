📊 대시보드 — 전체 영업 현황을 한눈에 파악하는 메인 화면입니다.
• KPI 카드: 올해 총 발주 건수, 활성 거래처, 이탈 거래처, 전체 누적 현황을 보여줍니다.
• 월별 히트맵: 어느 달에 발주가 집중되는지 색상 강도로 표시합니다. 오른쪽 검색창으로 특정 거래처를 찾아 해당 거래처의 월별 패턴만 볼 수 있습니다.
• TOP 발주처: 전체 기간 기준 발주량이 많은 거래처 순위입니다. 학교(초등·중학·고등·대학교·대학원), 관공서(도청·교육청·교육지원청) 유형별 필터링이 가능합니다. 표시 수도 직접 설정할 수 있습니다.

표시:
📅 이달의 예상발주 — 선제적 영업을 위한 핵심 기능입니다.
• 기준 연도·월을 선택하면 그 시기에 발주했던 모든 거래처 목록이 나타납니다.
• 탭 옆 숫자(배지)는 선택한 연도·월의 총 발주 건수입니다. 예: 2025년 3월 선택 시 → 2025년 3월에 들어온 총 건수 표시.
• 각 행의 올해 발주 열은 해당 발주처가 올해 발주한 총 건수를 보여줍니다. 미발주는 아직 올해 발주가 없는 곳, 발주됨 N건은 올해 이미 N건 발주한 곳입니다.
• '발주됨 N건'을 클릭하면 올해 실제로 어떤 건들이 발주됐는지 팝업으로 확인할 수 있습니다.
• ⚠️ '발주됨 N건'은 발주처(회사명) 기준으로 집계됩니다. 담당자·건명이 달라도 같은 발주처에서 온 모든 건을 합산합니다.
• 상단 필터 버튼으로 미발주만 또는 발주됨만 따로 볼 수 있습니다.
• 활용 방법: 기준 연도 2025, 월 3월 선택 → 작년 3월에 발주한 거래처 중 올해 아직 연락이 없는 곳을 확인하고 선제 영업 연락.
• 등급 기준: ⭐ VIP 전체 기간 발주 건수 상위 10%  |  일반 상위 10~50%  |  소규모 하위 50% (전체 기간 누적 발주 건수 기준 자동 분류)

발주 목록
⚖️ 연도 비교 — 두 연도의 발주 현황을 나란히 비교하는 전략 분석 기능입니다.
• 연도 A(기준)와 연도 B(비교)를 선택하고 '비교 분석' 버튼을 누르세요.
• 🔴 A에만 있음: A연도에 거래했지만 B연도에 발주가 없는 곳 → 잃어버린 거래처. 영업 복구 대상입니다.
• 🟢 B에만 있음: B연도에 새로 생긴 거래처 → 신규 개척 성과입니다.
• 건수 변화: 양쪽 모두 있는 거래처의 발주 건수 증감을 보여줍니다.
• 담당자 변동: 같은 거래처인데 연도 간 담당자가 바뀐 경우를 표시합니다. 관공서·공공기관은 인사이동으로 담당자가 교체되면 거래가 끊기는 경우가 많습니다. 변동이 있는 곳은 새 담당자에게 먼저 연락해 관계를 재정립하는 것이 중요합니다.
• 각 항목 클릭 시 해당 거래처의 전체 히스토리로 이동합니다.
연도 A vs 연도 B
🔴 A에만 있음 (B에서 사라진 거래처)
🟢 B에만 있음 (신규 거래처)
📊 건수 변화 (양쪽 모두 존재하는 거래처)
👤 담당자 변동 감지 (같은 거래처, 담당자가 바뀐 곳)
🏢 거래처 히스토리 — 특정 거래처의 전체 발주 이력을 연도별로 확인하는 기능입니다.
• 왼쪽 검색창에 발주처명 또는 담당자 이름을 입력하면 실시간으로 목록이 좁혀집니다.
• 목록에서 거래처를 클릭하면 오른쪽에 연도별 발주 이력, 담당자, 연락처, 이메일이 표시됩니다.
• 이탈이 표시된 거래처(빨간 점)는 올해 발주가 없는 곳입니다.
• 활용 방법: 거래처에 전화하기 전 검색 → "이 담당자가 작년에 어떤 건을 발주했지?"를 빠르게 확인하고 대화에 활용하세요.

📈 트렌드 분석 — 거래처별 발주 증감 흐름을 파악하는 기능입니다.
• 왼쪽 표에는 전년 대비 발주가 늘어난 거래처, 오른쪽에는 줄었거나 끊긴 거래처가 자동 분류됩니다.
• 아래 꺾은선 차트에서 보고 싶은 거래처 이름을 직접 클릭해서 선택·해제할 수 있습니다. 기본은 상위 6개입니다.
• [전체 켜기/끄기] 버튼으로 기본 상위 6개를 한번에 켜고 끌 수 있습니다.
• 발주처명을 검색하면 해당 거래처 중심으로 분석되고 차트 선택이 초기화됩니다.
• 표의 발주처명을 클릭하면 해당 거래처 히스토리로 바로 이동합니다.

거래처별 연도별 건수 추이 — 직접 선택
🔍 반복 건명 탐지 — 특정 키워드가 포함된 발주 건을 연도별로 추적하는 기능입니다.
• 검색창에 건명 키워드를 입력하고 탐지 버튼을 누르세요.
• 상단 연도별 건수 카드: 각 연도에 해당 키워드의 발주가 몇 건 있었는지 한눈에 비교합니다. 카드를 클릭하면 해당 연도 데이터만 필터링해서 볼 수 있습니다. 다시 클릭하면 전체로 돌아옵니다. 올해(★)가 몇 건인지 특히 확인하세요.
• 결과 내 추가 검색: 전체 매칭 목록이 많을 경우, 결과 내 검색창에서 발주처명·담당자·건명으로 다시 좁힐 수 있습니다.
• 키워드는 매칭된 부분이 주황색으로 강조됩니다. 각 행 클릭 시 거래처 히스토리로 이동합니다.
• 활용 방법: '학사력' 검색 → 연도 카드로 작년 건수 확인 → 올해(★) 카드 클릭해서 아직 발주 안 한 학교 파악 → 선제 연락. '홍보물', '보고서', '포스터', '현수막' 등 자주 쓰는 키워드 활용.

🚨 이탈 감지 — 4가지 분석 모드로 거래처 이탈·위험을 다각도로 파악합니다.
• ① 연도별 비교: 기준 연도 → 다음 연도 1:1 비교. 완전이탈·감소·신규를 분류합니다.
• ② 누적 이탈 추적: 특정 연도 이후 1년·2년·3년 이상 거래 없는 곳을 단계별로 구분합니다.
• ③ 위험도 스코어링: 발주 추이를 종합 분석해 🔴위험·🟡관찰·🟢안전으로 자동 분류합니다.
• ④ 복귀 거래처: 2년 이상 거래 공백 후 최근 다시 발주한 곳을 표시합니다.
• 모든 항목을 클릭하면 해당 거래처 히스토리로 바로 이동합니다.
